# Calculator Project

Developper: Sayed Shafaque Bin Nur, CSE, RUET


A modern, responsive calculator web application built with HTML, CSS, and JavaScript.

## Features

- **Basic Operations**: Addition, subtraction, multiplication, division
- **Advanced Functions**: Percentage calculations
- **Dynamic Font Sizing**: Text automatically adjusts to fit the display screen
- **Responsive Design**: Works seamlessly on different screen sizes
- **Error Handling**: Displays error messages for invalid operations
- **Backspace Function**: Delete last input character
- **Clear Function**: Reset calculator to initial state

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome Icons

## File Structure

```
calculator-project/
│
├── index.html          # Main HTML structure
├── style.css           # Styling and layout
└── script.js           # Calculator logic and functionality
```

## How to Use

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Click buttons to perform calculations
4. Use `C` to clear, `←` to delete last character, `=` to calculate result

## Key Functionalities

- **Input Handling**: Captures button clicks and builds mathematical expressions
- **Evaluation**: Uses JavaScript's `eval()` to compute results
- **Display Management**: Shows expressions and results with precision up to 9 decimal places
- **Font Adjustment**: Automatically shrinks text when it exceeds display width

## Browser Compatibility

Compatible with all modern browsers including Chrome, Firefox, Safari, and Edge.
